C. Обогащение контекста

Задача
Нужно «сжать» набор пользовательских запросов: объединить в один контекст те
запросы, чьи множества слов пересекаются хотя бы по одному слову (объединение
транзитивно). Требуется найти:
- количество получившихся контекстов;
- размер (в уникальных словах) самого большого контекста.

Формат ввода
1) В первой строке — число N (1 ≤ N ≤ 1000) — количество запросов.
2) Далее для каждого i = 1..N даны две строки:
   - Mᵢ (1 ≤ Mᵢ ≤ 1000) — число слов в i‑м запросе;
   - строка из Mᵢ слов (только строчные латинские буквы, длина 1..10),
     разделённых пробелами.

Замечание: одинаковые слова внутри одного запроса считаются одним и тем же словом.

Формат вывода
Выведите два целых числа через пробел:
- количество контекстов;
- размер (в уникальных словах) самого большого контекста.

Пример 1
Ввод
3
4
a four word prompt
2
small prompt
5
unique line with five words
Вывод
2 5

Пример 2
Ввод
1
1
a
Вывод
1 1

Пример 3
Ввод
2
2
a b
2
b c
Вывод
1 3
